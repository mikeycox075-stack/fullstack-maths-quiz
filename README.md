
A full-stack web application featuring user authentication, interactive quizzes, and score tracking.

- User signup and login (JWT authentication)
- Secure password hashing (bcrypt)
- Interactive quiz with random questions
- Score tracking saved to MongoDB
- Dashboard showing user performance


Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

Database:
- MongoDB Atlas

- JWT authentication
- Password hashing (bcrypt)
- Environment variables (.env)

1. Users create an account
2. Login generates a JWT token
3. Quiz loads 10 random questions
4. Score is calculated
5. Score is saved to database
6. Dashboard displays results
